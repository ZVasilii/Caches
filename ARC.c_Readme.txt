struct cell* fast_get_page(long long int page_name, struct list* T1, struct list* T2, struct list* B1, struct list* B2, int *p)	
Функция кеширования
page_name - название или индекс страницы
T1, T2, B1, B2 - инициализированные части кеша
p - вспомогательный параметр кеширования
Выдает указать на элемент кеша с данным названием(индексом)
Описание:
1)Поиск среди закешированных страниц
2)Если элемент не найден, то запускается стандартный поиск по памяти find_mem, после чего все равно кешируется
